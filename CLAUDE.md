# interpretability-research

Personal pet project (not MARE work): testing how much of Anthropic's "global workspace /
J-space" findings transfer to small open-weights models, via their open-sourced Jacobian lens.

**Session start: read `CONTEXT.md` first** (full handoff: story so far, verified findings,
environment), then `PLANS.md` (roadmap -- next up is experiment 02, the scale sweep).
`README.md` is the public-facing overview.

## Repo structure

- `experiments/NN-short-name/` -- one directory per experiment: `README.md` (writeup), code,
  own `requirements.txt`, and `results.md` (a BUILD ARTIFACT -- regenerated by the experiment
  script; commit the regenerated file together with the code change that produced it).
- `PLANS.md` -- update statuses in place when work starts/lands.
- `CONTEXT.md` -- append a short dated note when an experiment lands or a major decision is
  made, so the next session picks up cleanly.

## Hard constraints of this box (WSL2)

- **No CUDA.** Always install CPU torch first (`pip install torch --index-url
  https://download.pytorch.org/whl/cpu`) or pip pulls the ~2.5 GB CUDA wheel.
- **~7.6 GB RAM.** Load models in bf16. Qwen3.5-0.8B fits; 1.7B is borderline; 4B+ does not --
  bigger runs need a rented GPU (Colab/RunPod/vast.ai) on a personal account, NOT MARE AWS.
- Long commands (downloads, fits) via nohup + log polling; shells time out ~10 min.

## Working rules

- Pre-fitted lens checkpoints exist on HF `neuronpedia/jacobian-lens` for Qwen3/3.5/3.6 up to
  32B -- use them; only fit lenses yourself when that IS the experiment (see PLANS 03).
- Known negative result (exp 01): the hidden-intermediate multi-hop readout does not reproduce
  at 0.8B. Don't "fix" demos to force it; negative results are findings here.
- Upstream `jlens` is an unmaintained reference implementation -- solve breakage by pinning
  versions in the experiment's requirements.txt, not by patching upstream.
- Claims of results go through execution: run it, capture raw output in results.md, and keep
  the writeup consistent with what actually ran.
